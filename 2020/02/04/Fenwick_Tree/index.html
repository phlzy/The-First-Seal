树状数组

Created2020-02-04Updated2020-02-04

简介：

树状数组（二叉索引树，Fenwick树）的结构与线段树相似，功能上可以一些处理区间修改查询问题。相比线段树，树状数组的优点在于代码量少，操作方便，效率高于线段树，但它的缺点在于一般只能单点修改且用途不如线段树广泛。

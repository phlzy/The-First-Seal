CF1295B题解

Created2020-02-04Updated2020-02-10

链接：CF1295B Infinite Prefixes 洛谷CF1295B

  • 数学 字符串 *1700

    思路：

    由于可以无限拼接输入的01串，前缀和一定会周期性变化。先记录一个周期的所有前缀和，又因为空串也算是一个前缀，所以记录01串时从下标$1$开始，如图：
下标 $i$ 0 1 2 3 4 5 6
01串 $m_i$ / 0 1 0 0 1 0
前缀和 $s_i$ 0 1 0 1 2 1 2

所以每拼接一个完整的串，前缀和增加$2$，第$k$个前缀和$s_k$也可以算出：

对于$s_{len}$，若其不为0，则从$s_1$遍历到$s_len$，当$x-s_i$是$s_{len}$的正整数倍时，说明找到了一个答案。如果$x=0$则还需要再算上最前面空的串。若$s_{len}=0$，就只需要在$s_0$到$s_{len}$中寻找，若有$s_i$ 与$x$相等，那么在接上任意长度的整串后$s_{i+k*len}$也与$x$相等，即有无数答案；若找不到，说明无解。

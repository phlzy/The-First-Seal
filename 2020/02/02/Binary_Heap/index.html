二叉堆

Created2020-02-02Updated2020-02-02

简介：

(二叉)堆是一个数组，类似于一棵完全二叉树，树上的每一个节点对应数组中的一个元素。除了树的最底层，该树是完全充满的，而且是从左向右填充。表示堆的数组通常需要给出数组长度lenlenlen和包含的有效数据个数(堆中数据个数)sizesizesize两个属性。显然，有0≤size≤len0\le size\le len0≤size≤len。
对于下标iii指向的元素，可以算出它的父节点、左孩子和右孩子的下标分别为⌊i/2⌋\lfloor i/2\rfloor⌊i/2⌋、2i2i2i和2i+12i+12i+1。

堆可以分为两种，大根堆和小根堆。顾名思义，大根堆的最大元素存放在根节点，且在任意子树中子树包含的节点的值都不大于该子树的根节点的值。小根堆的存放方式正好相反。

模板题(P3378)简述：

初始小根堆为空，支持以下3种操作：

输入111 xxx：将xxx插入到堆中；

输入222：输出堆中最小数；

输入333：删除堆中最小数。

题解：

1、手写堆

只需要实现插入元素和删除堆顶两个功能，插入时将新元素放到堆底，并逐层与所在的子树的根节点相比较，若小于根节点则与其交换位置；删除操作与插入相反，先用堆底元素替换堆顶，再逐层向下比较，将原堆底元素交换到它应该在的位置。

2、STL优先队列

STL的优先队列(priority_queue)就是一个堆，但是它默认是一个大根堆，需要手动改成小根堆或者将输入变为相反数，输出时再反一次。

3、pb_ds库的堆(优先队列)

pb_ds：Policy-Based Data Structures

pb_ds库是GNU-C自带的一个C扩展库，封装了哈希表，平衡二叉树，字典树，堆等数据结构。

该库只在Linux下可以用！

其中的优先队列用法与STL的基本相同，包括 size()，empty()，push()，top()，pop()，clear()。

定义时由于名称与STL的相同，要带上命名空间：

Tag表示使用堆的类型，有配对堆（pairing_heap_tag）、二叉堆（binary_heap_tag）、二项堆（binomial_heap_tag）、经改良的斐波那契堆（thin_heap_tag）、冗余计数二项式堆（rc_binomial_heap_tag）等。

更多：可以用 begin() 和 end() 来获取 iterator 从而遍历；

可以 increase_key，decrease_key 和删除单个元素；

可以合并。

在不开O2优化的情况下，pb_ds库的二叉堆效率高于STL的优先队列。

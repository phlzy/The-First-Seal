洛谷P6103题解

链接：P6103

题解：

很明显这是一道DP题，并且对 $2^{64}$ 取模也就相当于用unsigned long long保存数据直接算，但是状态转移方程的设计很麻烦，因为有以下这些复杂的转换关系：

  • 语句(sentc) + 程序片段(ans) = 程序片段(ans)；

  • 程序片段(ans) + 2 = 语句块(blk)；

  • 语句块(blk) = 语句(sentc)；

  • 语句块(blk) +2 或 +4 是函数(func)；

  • 函数(func) +4 还是函数(func)；

  • 值(val) +2 还是值(val)，值(val) + 1 = 语句(sentc)；

  • 语句块(blk)是语句(sentc)，但语句(sentc)不是语句块(blk)；

  • 函数(func)都是值(val)，但值(val)不一定是函数(func)。

所以可以得到以下这些递推式：

  • $blk_i=ans_{i-2}$

  • $func_i=func_{i-2}+blk_{i-2}+blk_{i-4}$

  • $val_i=val_{i-2}+func_i$

  • $sentc_i = val_{i-1} + blk_i$

  • $ans_i=\sum_{k=0}^{i-1} {(ans_k\times sentc_{i-k})}$

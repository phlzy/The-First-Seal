HTML5基础知识

Created2020-07-17Updated2020-09-09Visited

HTML即Hyper Text Markup Language（超文本标记语言），目前主流浏览器均支持HTML5。以下是一些HTML5最基本的概念。

常用语句：

  • 声明：<!DOCTYPPE>

    • HTML5：<!DOCTYPPE html>

  • <head>

    • <title> 显示网页标签的名字
    • <link rel="stylesheet" type="text/css" href=""> 外部样式表
    • 内部样式表：

  • <body>

    • 标题 <h1> 到<h6>逐渐变小

    • <p> 段落（结束后另起一行）

    • <a> 超链接，用法：

      <a href="https://phlzy.github.io/">blog</a>

    • <img> 图片，用法：

      <img src="images/html.png">

    • <br/> 空标签（间距较小）

    • 格式化：

    • 内联样式表：<a style="color: aqua">

元素标签

块：结束后会换行

标题 <h1> 等，段落 <p> 及列表 ul 等均为块元素

内联元素：结束后一般不会换行

div元素：一般配合CSS样式作为组合HTML元素的容器使用

span元素：内联元素，文本的容器

【专题笔记#2】库函数与STL

Created2020-11-07Updated2020-12-06Visited

写几个容易错的地方，希望以后不要再错在这种东西上面了。

memset

众所周知，memset 的复杂度是 $O(n)$ 的，之前上新生课的时候我还提到过，结果回头打cf的时候就忘了，喜提TLE。

以及 memset 的第二个参数一般取以下四种：

  1. memset(dp, 0, sizeof(dp))，初始化为 $0$
  2. memset(dp, -1, sizeof(dp))，初始化为 $-1$
  3. memset(dp, 0x3f, sizeof(dp))（$63$），初始化为 $1061109567$（0x3f3f3f3f）
  4. memset(dp, ~0x3f, sizeof(dp))（$-64$），初始化为 $-1061109568$（~0x3f3f3f3f）

multiset

虽然和 set 很像，但是用的时候还是要稍微动动脑子的：

set 是普通的集合，erase 一个元素和对应的迭代器效果是一样的；

multiset 是可重集，erase 一个元素会删除所有与其相等的，而删除对应的迭代器只会删掉一个。

这就是P1286 两数之和改了8次才对的原因之一。

pb_ds

仅作了解就可以了。里面红黑树和可并堆应该是挺厉害的，但是 trie 和 splay 就很菜。

用起来也很麻烦。以后有空再去看吧。

其实一定要用平衡树和可并堆的情况并不多，很多时候只是没想到正解然后直接上平衡树（比如今年的CSP-J T2）。

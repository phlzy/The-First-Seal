后记

cmd的问题

出现了名为 Spawn failed 的错误，似乎是找不到仓库了，上网查了一下解决方案，发现问题出在Windows的cmd上面。

解决方法如下：

  1. 打开Git，输入 cd 和 .deploy_git 文件夹的路径，打开这个文件夹
  2. 输入 git log 查看日志
  3. 输入 q 退出日志状态
  4. 找到对应的版本id，输入 git reset --hard id，或者根据顺序用 git reset --hard HEAD^ 来回退到上一个正常的版本
  5. hexo clean，hexo g，hexo d 正常更新一遍
  6. 抛弃cmd，以后使用Git进行更新
